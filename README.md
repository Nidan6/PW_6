Опис проєкту
Цей проєкт являє собою багатосторінковий веб-сайт туристичної компанії, яка спеціалізується на турах та екскурсіях по Україні. Сайт розроблено як практичну роботу для демонстрації навичок верстки за допомогою HTML5 та CSS3.

Сайт має зручну навігацію, приємний дизайн та містить базову інформацію, необхідну для ознайомлення з послугами агенції.

Структура файлів
Проєкт складається з 4-х HTML-сторінок та одного загального файлу каскадних таблиць стилів (CSS):

main.html — Головна сторінка. Включає опис туристичних пропозицій, список популярних напрямків цього сезону та блок з цікавими фактами про Україну.

tours.html — Тури та ціни. Містить таблицю з розкладом майбутніх турів (назва, тривалість, дата початку та ціна).

about.html — Про нас. Сторінка з короткою історією компанії та переліком основних переваг (кількість екскурсій, унікальні маршрути, нагороди гідів).

contacts.html — Контакти. Сторінка з контактними даними (адреса, email) та інтерактивною формою зворотного зв'язку для користувачів.

styles.css — Файл стилів. Забезпечує єдиний візуальний стиль для всіх сторінок сайту, включаючи налаштування кольорів, шрифтів, стилізацію таблиць, форм та навігаційного меню.

Використані технології
HTML5: Використано сучасні семантичні теги (header, main, section, article, aside, nav, footer) для правильної структуризації контенту.

CSS3: * Використання лінійних градієнтів (для header).

Flexbox для вирівнювання елементів навігаційного меню.

Псевдокласи (наприклад, :hover для посилань та кнопок).

Базова адаптивність медіа-запитами (Media Queries) для коректного відображення на різних пристроях.

Google Fonts: Підключено сторонній шрифт "Open Sans" для покращення типографіки сайту.
